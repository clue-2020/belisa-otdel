МЕТОДИЧЕСКИЙ ИНСТРУМЕНТАРИЙ
П.1 Компетенции информанта (информант - лицо, включенное в эксперимент):
П.1.1 занятие (8 индикаторов);
П.1.2 знания (6 индикаторов);
П.1.3 приоритеты (9 индикаторов);
П.1.4 навыки (11 индикаторов).

П.2 Частная позиция информанта:
П.2.1 оптимистическая - 1;
П.2.2 нейтральная - 0,366;
П.2.3 пессимистическая - 0,134;
П.2.4 нет оснований для оценки - 0.

П.3 К-т принадлежности к возможному исходу:
П.3.1 = ∑ (П.2.1-П.2.4) / 8
П.3.2 = ∑ (П.2.1-П.2.4) / 6
П.3.3 = ∑ (П.2.1-П.2.4) / 9
П.3.4 = ∑ (П.2.1-П.2.4) / 11 * фактический опыт выполнения заданий за последние 5 лет / 5 (макс.)
Максимальный результат - 1.

П.4 Для определения коэффициента доверия (V) информанта используется критерий умеренного оптимизма Гурвица: V = П.3.4 * макс (П.3.1; П.3.2; П.3.3) + (1-П.3.4) * мин (П.3.1; П.3.2; П.3.3)
Максимальный результат - 1.

ЭЛЕКТРОННАЯ ФОРМА АНАЛИТИЧЕСКАЯ ПАНЕЛЬ

Оценка прогрессивности и полезности результатов научно-исследовательского труда на стадии планирования, выполнения и завершения темы НИР 

 МЕТОДИЧЕСКИЙ ИНСТРУМЕНТАРИЙ ОЦЕНКИ ПРОГРЕССИВНОСТИ (ПРИОРИТЕТНОСТИ) И ПОЛЕЗНОСТИ (СЕРЬЕЗНОСТИ) РЕЗУЛЬТАТОВ НАУЧНО-ИССЛЕДОВАТЕЛЬСКОГО ТРУДА

1) Электронная заявка является основанием для включения темы НИР в темплан НИР организации.
2) Инициаторы тематических исследований формируют постоянно действующую респондентскую интернет-сеть темплана НИР для непрерывного обмена информацией и знаниями.
3) Электронная заявка состоит из 2 разделов (сценарных планов): F1 – основания; F2 - действия. Каждый сценарный план включает определенное количество компонент (N): F1 (N) = 4; F2 (N) = 3. Каждый компонент содержит определенный перечень вопросов (n).
4) Оценка инициатора и участников темы НИР имеет 4 уровня позиционирования: оптимистическая оценка (1 голос), нейтральная оценка (0,366 голоса), пессимистическая оценка (0,134 голоса), уровень полной неопределенности или отсутствия оснований для частной позиции (0 голосов).
5) Оценка сценарных планов электронной заявки осуществляется по двум атрибутам:
5.1 прогрессивности (приоритетности оснований) - Кп.
5.2 полезности (серьезности воздействия на проблему, дефект, противоречие) - Кс.
6) Определение Кп:
Структура научно-исследовательского труда (компоненты)*:
N.1 познавательный труд (20%);
N.2 творческий труд (32%);
N.3 формально-технический труд (40%);
N.4 управленческий труд (8%).
* научно-исследовательский труд является дескриптивным множеством
Частная позиция информанта:
P.1 оптимистическая - 1;
P.2 нейтральная - 0,366;
P.3 пессимистическая - 0,134;
P.4 нет оснований для оценки - 0.
Коэффициент принадлежности (r) частной позиции информанта к максимальному результату: сумма частных позиций информанта для каждой компоненты разделить на количество вопросов (n).
Коэффициент прогрессивности (приоритетности) научно-исследовательского труда: Кp= N.1 * R1 + N.2 * R2 + N.3 * R3+ N.4*R4. Максимальное значение Кп: 1.
7) Определение Кс:
Компоненты результатов научно-исследовательского труда (компоненты):
N.5 научные результаты
N.6 уникальность и визуализация научного контента
N.7 прикладная ценность
Частная позиция информанта:
P.1 оптимистическая - 1;
P.2 нейтральная - 0,366;
P.3 пессимистическая - 0,134;
P.4 позиция отсутствует - 0.
Коэффициент принадлежности (r) частной позиции информанта к максимальному результату: сумма частных позиций информанта для каждой компоненты разделить на количество вопросов (n).
Коэффициент полезности (серьезности) научно-исследовательского труда (Кс)
Для оценочной функции Кс использован критерий умеренного оптимизма Гурвица:
Кс = 0.5 * max(N5*r,N6*r,N7*r) + (1-0.5) * min(N5*r,N6*r,N7*r)
Максимальный результат - 1.

АНАЛИТИЧЕСКАЯ ПАНЕЛЬ

Установление размеров надбавки за специфику работы в бюджетной научной организации

МЕТОДИЧЕСКИЙ ИНСТРУМЕНТАРИЙ
Электронная заявка заполняется руководителем структурного подразделения, чтобы установить размер планируемой надбавки за специфику работы и сумму вознаграждения работнику структурного подразделения с учетом:
- частной позиции руководителя структурного подразделения (e): оптимистическая позиция имеет вес, равный 1,0 баллу; нейтральная - 0,366 баллов; пессимистическая - 0,134 балла; позиция отсутствует - 0 баллов;
- критериев установления размеров надбавки за специфику работы (F) согласно приказу от 19.02.2020 №28;
- профессиональных и творческих * достижений работника за прошлый год - E (по умолчанию Е = 1);
структуры научно-исследовательского труда * - K;
- литита бюджетных средств, обслуженных плановыми сметами на выполнение работ - L.
* для категории научных работников

Принятые пределы размеров надбавки за специфику работы:
- Критерий F1 предусматривает надбавку до 50% для списка из 4 видов работ (f1 = 4), хотя бы один вид работы должен иметь оценку больше 0);
- Критерий F2 предусматривает надбавку до 100% для списка из 3 видов работ (f2 = 3), хотя бы один вид работы должен иметь оценку больше 0, f1> 0);
- Критерий F3 предусматривает надбавку до 150% для списка из 7 видов работ (f3 = 7), хотя бы один вид работы должен иметь оценку больше 0, f2> 0);
- Критерий F4 предусматривает надбавку до 200% для списка из 7 видов работ (f4 = 7), хотя бы один вид работы должен иметь оценку больше 0, f3> 0).

Структура научно-исследовательского труда (компоненты) *:
N1 познавательный труд (20% рабочего времени);
N2 творческий труд (32% рабочего времени);
N3 формально-технический труд (40% рабочего времени);
N4 управленческий труд (8% рабочего времени).
* структура научно-исследовательского труда, предложенная А.Г.Чернявским

Коэффициент прогрессивности (приоритетности) научно-исследовательского труда определяется по формуле: К = 0,2*e + 0,32*e + 0,4*e + 0,08*э. Максимальное значение К - 1.

ПРН = (50*сумма f1 + 50*сумма f2 +50*сумма f3 + 50*сумма f4) * Е
Максимальный ПРН: 200хЕ, где E принимает значение от 1 (по умолчанию) до 2.

Размер материального вознаграждения:
1) Размер материального вознаграждения работников структурного подразделения осуществляется в пределах бюджетных бюджетных средств, предусмотренных плановыми сметами на выполнение работ (L).
2) Распределение L между работниками структурного подразделения осуществляется пропорционально S, где S = ПРН / 200 * К
